# PHP服务器启动说明

## 方式一：使用PHP内置服务器（推荐）

### 1. 安装PHP

**方法A：下载PHP独立版本**
1. 访问 https://windows.php.net/download/
2. 下载最新版本的 PHP（选择 Thread Safe 版本，ZIP格式）
3. 解压到 `C:\php` 目录
4. 将 `C:\php` 添加到系统环境变量 PATH 中

**方法B：使用XAMPP（最简单）**
1. 访问 https://www.apachefriends.org/zh_cn/index.html
2. 下载并安装 XAMPP
3. XAMPP会自动安装PHP、Apache、MySQL等
4. PHP路径通常在 `C:\xampp\php\`

### 2. 启动服务器

打开命令行（PowerShell或CMD），进入项目目录，运行：

```bash
cd C:\Users\Administrator\Downloads\def11111\def
php -S localhost:8000
```

或者指定完整路径（如果PHP未添加到PATH）：
```bash
C:\xampp\php\php.exe -S localhost:8000
```

### 3. 访问页面

在浏览器中访问：
- 登录页面：http://localhost:8000/login.tpl.php
- 注册页面：http://localhost:8000/register.tpl.php

## 方式二：使用XAMPP控制面板

1. 安装XAMPP后，打开XAMPP控制面板
2. 启动 Apache 服务
3. 将项目文件复制到 `C:\xampp\htdocs\def\`
4. 访问：http://localhost/def/login.tpl.php

## 方式三：使用WAMP

1. 下载并安装 WAMP Server
2. 启动 WAMP 服务
3. 将项目文件复制到 `C:\wamp64\www\def\`
4. 访问：http://localhost/def/login.tpl.php

## 注意事项

- 确保PHP版本 >= 7.0
- 如果使用内置服务器，PHP代码会正常执行
- 验证码功能需要后端支持 `index.codeimage` 路由
